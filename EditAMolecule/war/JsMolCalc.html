Information for edX

The GWT project can be found here.

  • Raw Material provided by Biologists:
    • .mol file for each of a set of molecules.
    • Brief text descriptions of each molecule.
  • Transforming Raw Material into a Problem done by edX:
    1. Randomly choose a .mol file from the set. Currently there's only one in the same directory as this file. You will probably need to arrange this differently.
    2. Load the description of the molecule into the page where it now reads "The molecule Dopamine is shown below. Dopamine is a neurotransmitter".
    3. Load the .mol file into JsMolCalc. Currently, the java code reads a file called "dopamine.mol". This is done in edu.umb.bio.jsMolCalc.client.JsMolCalc.java lines 71-87. You will need to make this something that fits with your randomization code.
    4. JsMolCalc then will automatically display the molecule, calculate it's properties (the correct answers), and set up a set of appropriate tasks
    5. As the student completes various tasks, the entries in the "taskChooser" get "COMPLETED" appended to their names; you can score on the basis of this.
