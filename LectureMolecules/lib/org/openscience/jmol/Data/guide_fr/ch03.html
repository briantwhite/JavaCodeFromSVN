Le menu Fichier

Ouvrir

Ouvre des fichiers dans Jmol. Jmol peut lire des fichiers des types suivants:

  • ABINIT

  • ACES II

  • ADF - Amsterdam Density Functional

  • CML v1.0 and 1.0.1

  • GAMESS

  • Gaussian 92/94/98

  • Ghemical MM

  • Jaguar

  • MDL molfile

  • PDB (ATOM and HETATM entries only)

  • XYZ (single and multiple frame)

Fichiers Récents...

Ouvre une boîte de dialogue qui contient une des fichiers ouverts récemment oar le programme. Le fichier peut être ouvert à nouveau en le sélectionnant et en cliquant sur le bouton Ouvrir.

Script...

Ouvre une fenêtre où des commandes de script Rasmol peuvent être exécutées. RasMol/Chime Scripts contient des informations sur les scripts RasMol/Chime.

Enregistrer Sous...

Exporter

Exporter -> Exporter une Image...

Jmol peut exporter des images des types suivants:

  • BMP

  • GIF

  • JPG

  • PNG

  • PPM

Il faut noter que les images GIF ne peuvent être exportées que si l'image affichée est en 8 bits.

Exporter -> Rendu dans pov-ray...

Pour des images de haute qualité, Jmol s'interface avec le ray tracer POV-Ray. Pour utiliser cette possibilité, vous devez avoir l'utilitaire en ligne de commande povray installé. Vous pouvez le télécharger depuis www.povray.org.

Imprimer

Des fichiers Postscript peuvent être exportées directement vers l'imprimante ou vers un fichier.

Quitter

Quitte cette session Jmol.
