Le menu Edition

Tout sélectionner

Sélectionne tous les atomes dans l'écran de visualisation.

Tout Désélectionner

Désélectionne tous les atomes dans l'écran de visualisation.

Créer un cristal...

Ouvre une boîte de dialogue qui permet de construire un nouveau cristal en partant du fichier actuel.

Cette boîte de dialogue n'est active que si le fichier ne contient pas d'information à propos de la structure du cristal. Une fois que le cristal a été créé, l'option Créer un critsal est désactivée mais les informations de structure du cristal peuvent être obtenues en passant par la boîte de dialogue Propriétés du cristal dans le menu Extra.

Les boîtes de dialogue Créer un Cristal dans le menu Edition et Propriétés du Cristal dans le menu Extra sont les mêmes mais leur comportement est légérement différent suivant d'où elles ont été appelées. Leur description est donnée dans la section intitulée “Propriétés du Cristal...” plus loin dans ce manuel.

Propriétés d'Atome...

Ce menu ouvre une boîte de dialogue dans laquelle des propriétés atomiques peuvent être ajoutées à un grand nombre de types d'atomes. Ces propriétés incluent les couleurs utilisées pour décrire les atomes.

Options...

Ce menu ouvre une boîte de dialogue pour modifier les paramètres et laisser l'utilisateur personnaliser ses options. Ces réglages sont sauvegardés d'une session à l'autre.

Onglet Affichage

Dans cet onglet quelques réglages d'affichage peuvent être modifiés. Par défaut, l'anti-aliasing est désactivé. L'activer améliore la qualité des images de facon considérable mais réduirea aussi les performances.

Remarque: La JVM 1.3.1 qui est livrée avec Macintosh OS X et OS X 10.2 (Jaguar) a des anomalies fatales avec certaines opérations d'interpolation de pixle et d'anti-aliasing. En conséquence, vous pourriez être obligé de désactiver l'utilisation de Java Graphics2D sur cette plate-forme.

Remarque: La JVM 1.4.1 qui a été distribuée par Apple semble avoir résolu quelques problèmes pour Jmol ... mais en a créé *beaucoup* d'autres :(

Onglet Couleurs

Dans cet onglet, des couleur générales d'affichage peuvent être modifiées. Par exemple, la couleur de fond et la couleur utilisée pour mettre en surbrillance les atomes sélectionnés.
